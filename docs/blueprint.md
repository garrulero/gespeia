# **App Name**: ChatGemini

## Core Features:

- Message Input: Allows the user to input messages as if they were in a WhatsApp chat.
- Gemini Integration: Connects to the Gemini API to generate responses based on user input.
- Dynamic Chat Interface: Displays user messages and Gemini-generated responses in a real-time chat interface, similar to WhatsApp.
- Response Selection Tool: Uses an AI tool to determine when to offer alternative response options for the user to choose from if they aren't satisfied with Gemini's first response.

## Style Guidelines:

- Primary color: Deep teal (#008080), evoking reliability and clear communication.
- Background color: Light teal (#E0F8F8), creating a soft, unobtrusive backdrop.
- Accent color: Muted blue (#4682B4) for subtle emphasis and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Use simple, outlined icons in a monochromatic style to maintain a clean and consistent look.
- Incorporate subtle animations for message arrival and sending confirmations to enhance the user experience.
